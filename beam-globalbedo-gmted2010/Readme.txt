Installation of GMTED2010_30
==============================
create file:
$HOME/.beam/beam-core/auxdata/dem/GMTED2010_30/dem.properties

with a line containing the installation directory
---------------------------------------
dem.installDir=/${INSTALL_PATH}/GMTED2010_Mean_30arsecs
---------------------------------------
Where ${INSTALL_DIR} is the directory
into which you unpacked the GMTED2010_Mean_30arsecs.zip file.
